### 1.5
El prompt que me dió buenos resultados fue el primero que realicé ya que iba con indicaciones bastantes claras de lo que quería, aunque de todas formas tuve que cambiarle y agregarle una cosas:
        
        "Crea el CSS base para una Landing Page de e-commerce. Usa CSS Grid para organizar el catálogo de 3 productos para que sea responsivo (1 columna en móvil, 3 en PC). Diseña botones atractivos con un color de acento fuerte (Call to Action) y efectos :hover. Usa una paleta de colores minimalista y moderna, asegurando un contraste AAA para la lectura de los precios. pero quiero que el diseño lo hagas segun lo que sabes de mi, es decir como me gusta trabajar y que se apegue mejor a mi estilo sin olvidar lo más importante de la pagina: que sea llamativo pero bastante simple y que venda bien la idea de los productos"

Los principios de UX que usé para los botones de *Añadir al carrito*, fue agregar un buen espaciado entre el texto y el borde así como un buen contraste en los colores con el mismo fin: que fuese legible.

### 2.6

Para poder cambiar la dirección de la imagen y el texto porque está diseñado para organizar elementos en una sola dimensión (filas o columnas). Esto permite cambiar fácilmente el lugar del contenido adptándolo a distintos tamaños de pantallas y es mucho más sencillo en comparación con Grid.
